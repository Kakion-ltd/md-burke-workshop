# MD Burke Ltd Workshop Management System v2.0

A complete workshop management solution for agricultural machinery repair businesses.

## Features

- **Job Management** - Full lifecycle from booking to collection
- **Real-time Timer** - Track labour hours per job
- **Parts Inventory** - Stock tracking with Park Mode
- **Invoice Generation** - Professional invoices with VAT
- **Customer Database** - Contact history and machine records
- **Supplier Management** - Track suppliers and orders
- **Calendar View** - Schedule visualization
- **Reports** - Revenue and performance analytics
- **Dual Interface** - Desktop and mobile views

## Deploy to Netlify

1. Push this folder to a GitHub repository
2. Connect to Netlify
3. Netlify will auto-detect the build settings

Or drag & drop the folder to Netlify's deploy page.

## Local Development

```bash
npm install
npm start
```

## Tech Stack

- React 18
- Tailwind CSS
- Lucide React Icons
